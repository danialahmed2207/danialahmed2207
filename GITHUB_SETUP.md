# 🚀 So veröffentlichst du dein GitHub Profil-README

## Schritt 1: Repo auf GitHub erstellen

1. Gehe zu [github.com/new](https://github.com/new)
2. **Repository-Name:** `danialahmed2207` (exakt dein GitHub-Username!)
3. **Beschreibung:** `GitHub Profile README`
4. Setze das Repo auf **Public**
5. **NICHT** "Initialize this repository with a README" ankreuzen
6. Klicke auf **Create repository**

## Schritt 2: Push von lokal

Nachdem das Repo erstellt ist, kopiere diese Befehle und führe sie im Terminal aus:

```bash
cd ~/github-profile-readme
git remote add origin https://github.com/danialahmed2207/danialahmed2207.git
git push -u origin main
```

## Schritt 3: Ergebnis prüfen

- Öffne `https://github.com/danialahmed2207`
- Die README erscheint jetzt ganz oben auf deinem Profil! 🎉

Folgende Repos sollten umbenannt werden (wirken unseriös):

| `mein-digitales-notizbuch` | `notes-app-python` oder löschen | Umbenennen oder Private |
| `charakter-captain-blaub-r` | `charakter-generator-python` oder löschen | Umbenennen oder Private |

### So benennst du ein Repo auf GitHub um:

1. Öffne das Repository auf GitHub
2. Klicke auf **Settings** (oben rechts)
3. Scrolle zu **Repository name**
4. Gib den neuen Namen ein
5. Klicke auf **Rename**

> ⚠️ **Tipp:** Wenn die Repos nicht repräsentativ sind, stelle sie auf **Private** oder lösche sie.
